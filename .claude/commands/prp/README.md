# PRP Commands

Product Requirements Planning (PRP) workflow commands for structured development planning.

## Commands

| Command | Description |
|---------|-------------|
| `/prp:prp-plan` | Create a PRP plan for a feature or project |
| `/prp:prp-prd` | Generate a Product Requirements Document |
| `/prp:prp-implement` | Implement features from a PRP plan |
| `/prp:prp-review` | Review PRP implementation against requirements |
| `/prp:prp-debug` | Debug issues identified during PRP review |
| `/prp:prp-issue-investigate` | Investigate issues related to a PRP |
| `/prp:prp-issue-fix` | Fix issues identified in PRP investigation |
