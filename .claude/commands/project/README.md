# Project Commands

Project creation, setup wizard, and validation commands.

## Commands

| Command | Description |
|---------|-------------|
| `/project:new-project` | Create a new project from scratch |
| `/project:project-create` | Create a project with template selection |
| `/project:project-wizard` | Interactive project setup wizard |
| `/project:validate` | Validate project configuration and structure |
| `/project:wizard-resume` | Resume an interrupted wizard session |
| `/project:wizard-status` | Check project wizard progress |
