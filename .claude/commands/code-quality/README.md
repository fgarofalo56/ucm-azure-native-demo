# Code Quality Commands

Code review and refactoring commands for Python and TypeScript codebases.

## Commands

| Command | Description |
|---------|-------------|
| `/code-quality:refactor-simple` | Quick refactoring check for Python/general code |
| `/code-quality:review-general` | General code review for the codebase |
| `/code-quality:review-staged-unstaged` | Review files in the git staging area |
| `/code-quality:TS-review-general` | TypeScript/Astro codebase review |
| `/code-quality:TS-review-staged-unstaged` | Review staged/unstaged TypeScript files |
